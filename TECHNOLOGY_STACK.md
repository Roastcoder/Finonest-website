# Finonest Technology Stack Documentation

## Project Overview
Finonest is a comprehensive loan management and financial services platform built with modern web technologies. The platform serves as a Direct Sales Agency (DSA) connecting customers with 50+ banking partners for various loan products.

## Frontend Technologies

### Core Framework & Build Tools
- **React 18** - Modern JavaScript library for building user interfaces
- **TypeScript** - Type-safe JavaScript for better development experience
- **Vite** - Fast build tool and development server
- **React Router DOM** - Client-side routing for single-page application

### UI Framework & Styling
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **shadcn/ui** - High-quality, accessible React components built on Radix UI
- **Radix UI** - Low-level UI primitives for building design systems
- **Lucide React** - Beautiful & consistent icon library

### State Management & Data Fetching
- **TanStack Query (React Query)** - Powerful data synchronization for React
- **React Hooks** - Built-in state management with useState, useEffect, etc.

### Form Handling & Validation
- **React Hook Form** - Performant, flexible forms with easy validation
- **Custom validation logic** - For consent forms and loan applications

### SEO & Meta Management
- **React Helmet** - Document head management for SEO optimization

### UI Components & Features
- **Responsive Design** - Mobile-first approach with Tailwind breakpoints
- **Dark/Light Theme Support** - Theme switching capability
- **Toast Notifications** - User feedback with Sonner and custom toaster
- **Loading States** - Custom loading screens and skeleton components
- **Animations** - CSS transitions and transforms for smooth UX

## Backend Technologies

### Database & Backend Services
- **Supabase** - Backend-as-a-Service platform providing:
  - PostgreSQL database
  - Real-time subscriptions
  - Authentication & authorization
  - Row Level Security (RLS)
  - API auto-generation

### Database Schema
- **PostgreSQL** - Relational database with the following key tables:
  - `pages` - Dynamic page content management
  - `components` - Reusable UI components
  - `content_blocks` - Dynamic content blocks
  - `theme_settings` - Site-wide theme configuration
  - `media_library` - File and image management
  - `admin_users` - Admin authentication and roles

### API Architecture
- **RESTful APIs** - Auto-generated by Supabase
- **Real-time APIs** - WebSocket connections for live updates
- **Custom API functions** - Server-side logic for complex operations
